# PKF_TestTasks

В этом репозитории находятся 4 программы:  
1) Калькулятор, реализованный через WPF (ветка Calculator)
2) Клиент-серверный мессенджер, реализованный через TCP взаимодействие и WPF для клиентов (ветка TcpMessenger)
3) Интерфейс в Unity, через который можно взаимодействовать с объектами на сцене (ветка Interface)
4) Мини игра куб-лабиринт в Unity (ветка CubeMaze)
